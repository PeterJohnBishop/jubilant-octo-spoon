# jubilant_octo_spoon

A flutter playground for building multiple websites.

# pre-requisit accounts
- AWS
- Google
- Resend
- Heroku
- Netlify

# backend services

- basic email
- email templates
- user account management 
- cloud storage for content 
- payment and invoicing 
- TOTP authentication 
- QR code generation

# frontend services
- mapping and directions 
- digital business cards
- mobile app
